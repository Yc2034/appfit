# AppFit

Personal fitness tracker - React Native + Expo + Supabase

## Daily Development

```bash
# Start iOS simulator + run app
open -a Simulator && npm run ios

# Or just run (if simulator already open)
npm run ios
```

## Other Commands

```bash
npm run android          # Run on Android
npm start                # Expo Go (scan QR)
npx expo start -c        # Clear cache & restart
```
